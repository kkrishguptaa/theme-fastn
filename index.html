How to use this colour scheme

The importance of colour in a websiteâs overall look and feel is well known. The right colour scheme can evoke emotions, create visual interest, and direct a userâs attention to specific elements on a page. Thatâs why the ftd colour scheme framework provides an easy and powerful way to define colour schemes and apply them to your website.

To start, you can choose from existing colour scheme packages or create your own custom colour scheme. To apply a colour cheme package on top of your package, youâll need to import it into FASTN.ftd.

Option 1: for documentation templates like doc-site

For example, letâs say youâre using the page component from doc-site package and want to apply the <add color scheme name here> scheme package on top of it.

To add color scheme to your fastn doc-site. Edit your FASTN.ftd file and add color scheme dependency into it.

In the below example, we are using @xkrishguptaa's color scheme.

Add color scheme dependency into your FASTN.ftd file as shown in below example:
-- fastn.dependency: xkrishguptaa.github.io/theme-fastn
Lang:
ftd

Now modify FASTN/ds.ftd module which is already added inside your fastn package.

Import theme-fastn dependency into FASTN/ds.ftd
-- import: xkrishguptaa.github.io/theme-fastn
Lang:
ftd

Change -- component page colors property ftd.color-scheme colors: $ftd.default-colors with ftd.color-scheme colors: $theme-fastn.main

replace this line of FASTN/ds.ftd file:
-- ftd.color-scheme color-scheme: $ftd.default-colors
Lang:
ftd
with:
-- ftd.color-scheme color-scheme: $theme-fastn.main
Lang:
ftd
With just a few lines of code, you can dramatically change the look and feel of your website using the ftd colour scheme.

Option 2: for custom fastn projects

Add color scheme dependency into your FASTN.ftd file as shown in below example:
-- fastn.dependency: xkrishguptaa.github.io/theme-fastn
Lang:
ftd
Now modify ftd file and import theme-fastn into ftd file
-- import: xkrishguptaa.github.io/theme-fastn
Lang:
ftd

Now add $theme-fastn.main to your component, as shown in below example:

Copy below code snippet inside your ftd file, then deploy and test
-- example: This is example of types used from `theme-fastn`

-- component example:
caption title:
ftd.color-scheme color-scheme: $theme-fastn.main

-- ftd.text: $example.title
role: $inherited.types.heading-hero
color: $inherited.colors.text

-- end: example
